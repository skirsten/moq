# moq-video

Native video encoding/decoding for [Media over QUIC](https://github.com/moq-dev/moq).

Counterpart to [`moq-audio`](https://crates.io/crates/moq-audio). Currently a stub to reserve the name on crates.io; the codec API will follow the same shape as `moq-audio` once the video sample input/output story is settled.
