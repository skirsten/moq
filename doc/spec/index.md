# Specifications

Internet-Drafts for the MoQ protocol suite.

- [moq-lite](/spec/draft-lcurley-moq-lite): The core pub/sub transport protocol.
- [hang](/spec/draft-lcurley-moq-hang): A real-time conferencing protocol built on moq-lite.
- [Use Cases](/spec/draft-lcurley-moq-use-cases): How to use MoQ for live media delivery.

## IETF Standards

MoQ is being standardized by the [IETF MoQ Working Group](https://datatracker.ietf.org/group/moq/about/). The above are personal drafts that complement the official working group documents:

- [MoqTransport](/concept/standard/moq-transport)
- [MSF](/concept/standard/msf)
- [LOC](/concept/standard/loc)
